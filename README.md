# Escape the Abandoned Lab

## Description
Escape the Abandoned Lab is a text-based adventure game where the player wakes up inside a mysterious research facility with no memory. The goal is to explore different areas, make decisions, collect items, and escape safely while avoiding dangers like security drones and lab creatures.

Your choices affect the outcome, leading to multiple possible endings.

## How to Run the Game
1. Make sure you have Python installed on your computer.
2. Download or clone this repository.
3. Open the file in IDLE or any Python editor.
4. Run the main game file (e.g, 'text_adventure.py').
5. Follow the on-screen instructions and type your choices.

## Features
- Multiple decision paths
- Inventory systems
- Different locations to explore
- Multiple endings based on player choices
